## Installation

### Prerequisites

- [Node.js](https://nodejs.org)
- [pnpm](https://pnpm.io/installation#using-corepack)
- Optional: [Visual Studio Code](https://code.visualstudio.com/)

### Installing dependencies

```sh
pnpm install
```

## How to run

```sh
pnpm run dev
```
